# duygu_analizi

## 📌 Proje Hakkında
Bu proje, metin verilerinden duygu analizi ve spam tespiti yapmayı amaçlamaktadır.

## 📂 Veri Seti
- `spam.csv`: Spam ve ham (normal) mesajları içeren veri seti

## 🛠️ Kullanılan Teknolojiler
- Python
- Pandas
- Scikit-learn / NLTK (kullandıysan)

## 🚀 Kurulum ve Çalıştırma
```bash
git clone https://github.com/filizzkayya6-gif/duygu_analizi.git
cd duygu_analizi
pip install -r requirements.txt
python main.py
```

## 📊 Proje Adımları
1. Veri yükleme ve ön işleme
2. Metin temizleme
3. Model eğitimi
4. Sonuçların değerlendirilmesi
